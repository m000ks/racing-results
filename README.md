# Racing Results

Race result data for thoroughbred meetings across Melbourne venues.

## Venues

| Code | Venue |
|------|-------|
| RHIL | Rosehill Gardens Racecourse |
| RAND | Randwick Racecourse |
| Flemington | Flemington Racecourse |
| Caulfield | Caulfield Racecourse |

## File Format

### Excel files (`.xlsx`)
Named using the pattern `YYYYMMDDVVVV0.xlsx` — e.g. `20260117RHIL0.xlsx` is the meeting at Rosehill on 17 January 2026.

### CSV files (`.csv`)
Named using the pattern `YYYYMMDD-venue-rNN.csv` where `NN` is the race number — e.g. `20260328-flemington-r05.csv` is Race 5 at Flemington on 28 March 2026.

## Contents

- **Results/** — Individual race result files by meeting date and venue
- **jockey ytd.xlsx** — Jockey year-to-date statistics
